<h3>源码简介：</h3>
<p>非常简单的一款通用货到付款类实物商城源码，可以生成单品推广链接。图片上传为阿里云oss。<p>
<p>单品订单系统<p>
<p>能生成一个推广页面订单，后台可以直接上传产品图片，生成H5页面。<p>
<p>多连接订单系统<p>
<p>即可轻松管理N个推广页面订单，多个链接，无需增加额外的费用。<p>
<p>多品订单系统<p>
<p>即可轻松管理N个推广页面订单，后台可以智能区分不同页面的订单，无需增加额外的费用。<p>
